06 · Spacing

SPACING & RADII

A 4-px base. Spacing favours dense industrial layouts; radii stay near-zero — chrome, badges, and CTAs use 0–4 px corners. Soft 8-px rounding only for floating UI elements (scroll-to-top, modals).
